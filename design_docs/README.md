## 方針

### なぜこのアプリを作成するのか

- 市場にあるのはコーポレート向けの、「タスクを消化する」アプリ
→日々のタスクではなく目標を達成するためのアプリを作成したい
- タスクや勉強、プロジェクトの進捗状況と計画のギャップを可視化したい

### 機能

- v1
    - ユーザ認証
    - タスクの表示
        - 自分が作成者のタスクのみを表示
    - タスクの作成・編集・削除
        - 自分が作成者のタスクのみ編集削除が可能
        - tasksの作成編集削除
        - daily_plansはtasksに紐づいて作成される
    - 実績入力
    - 実績と計画の可視化
- v2以降
    - サブタスクの紐づけ
    - レポート出力
    - タスク作成者以外の権限管理
- その後の追加機能
    - 通知機能
    - ai
    - モバイル連携
    - google calender連携（タスクの開始終了日時のみ連携可能か）
    - チャット機能とLINE連携
