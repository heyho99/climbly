### なぜこのアプリを作成するのか

- タスクや勉強、プロジェクトの進捗状況と計画のギャップを可視化したい
- タスク管理を簡単にしたい

### 機能

- マスト
    - ユーザ認証
    - タスクの作成
    - 日毎の計画入力（グラフいじる感じで）
    - サブタスクの紐づけ
    - 実績入力（時間と完了率）
    - 実績と計画の可視化
    - レポート出力
- 追加機能
    - 通知機能
    - ai
    - モバイル連携
    - google calender連携（タスクの開始終了日時のみ連携可能か）
